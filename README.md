# PhatcharaponInpor_PORTFOLIO
1. Robot soccer computer vision
เนื้อหาโดยย่อ:

เอกสารฉบับนี้สรุปเกี่ยวกับการพัฒนาระบบ Computer Vision สำหรับหุ่นยนต์เตะฟุตบอล โดยเน้นที่การประมวลผลภาพเพื่อระบุตำแหน่งของลูกบอล สนาม และคู่ต่อสู้

อธิบายถึงอัลกอริทึมที่ใช้ในการตรวจจับวัตถุเพื่อให้หุ่นยนต์สามารถตัดสินใจและเคลื่อนที่ในสนามได้อย่างแม่นยำตามสถานการณ์จริง

2. Join the Robocup Junior competition
เนื้อหาโดยย่อ:

รายงานสรุปประสบการณ์และการเข้าร่วมการแข่งขัน RoboCup Junior ซึ่งเป็นการแข่งขันหุ่นยนต์ระดับเยาวชนในระดับนานาชาติ

เนื้อหาครอบคลุมถึงขั้นตอนการเตรียมตัว การออกแบบหุ่นยนต์ และการแก้ปัญหาเฉพาะหน้าในระหว่างการแข่งขันเพื่อเพิ่มทักษะด้านวิศวกรรมและการทำงานร่วมกันเป็นทีม
